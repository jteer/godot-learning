## What is this?

Godot Roadmap is an interactive tracker for learning Godot 4 game development. Check off skills as you learn them, watch your progress fill up, and follow curated doc links for each topic ‚ÄĒ all saved locally in your browser.

---

## Features

- **5-phase 2D roadmap** ‚ÄĒ Foundations ‚Üí Core Mechanics ‚Üí Game Systems ‚Üí RPG Essentials ‚Üí Ship It
- **Per-skill doc links** ‚ÄĒ every checklist item links directly to the relevant Godot docs
- **Persistent progress** ‚ÄĒ saves to `localStorage`, no account needed
